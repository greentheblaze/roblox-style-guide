# Roblox Style Guide

A simplistic, community-driven extension of Roblox's Lua Style Guide.

---

This guide covers best practices to follow when programming on the Roblox platform. Visit the [home page][home-page] for more information.

## Contributing

Our project is an indefinite one, so if you think there are ways we could improve this resource, we request that you open up a new [pull request][pull-request]/[issue][issue] on our GitHub repository - feedback and contributions from the community is what keeps this project alive! Visit [CONTRIBUTING.md][contributing] for more information.
